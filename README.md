# Brand Management Simulation

An interactive educational simulation designed to teach brand management concepts through hands-on strategic decision-making. This simulation guides users through the complete brand lifecycle, from launch to crisis management, helping them understand the trade-offs and consequences of different branding strategies.

## Features

- **Three Strategic Phases**: Brand Launch, Extension, and Crisis Management
- **Real-time Metrics Tracking**: Monitor Brand Awareness, Customer Loyalty, Market Share, Profit Margin, and Brand Equity
- **Strategic Analysis**: Understand the implications and opportunity costs of each decision
- **Tactical Execution**: Select and implement specific tactics within chosen strategies
- **Educational Debrief**: Comprehensive feedback on performance and strategic insights

## Learning Objectives

- Understand the importance of positioning in brand strategy
- Learn the risks and rewards of brand extension
- Experience crisis management decision-making
- Appreciate the long-term value of brand equity
- Recognize the trade-offs between short-term gains and sustainable growth

## How to Play

1. **Strategic Phase**: Choose between competing strategies (e.g., Premium vs. Value positioning)
2. **Analysis Phase**: Review the consequences and opportunity costs of your choice
3. **Tactical Phase**: Select up to 2 specific tactics to implement your strategy
4. **Progress**: Advance through all phases to complete the brand lifecycle
5. **Debrief**: Receive final performance evaluation and strategic insights

## Technology

Built with React and styled with Tailwind CSS. Uses Lucide React for icons.

## Running the Simulation

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Deployment

This project is designed to be deployed as a GitHub Pages site. To deploy:

```bash
npm run build
npm run deploy
```

## Educational Context

This simulation is inspired by real-world brand management challenges and draws from marketing strategy frameworks. It's suitable for MBA students, marketing professionals, and anyone interested in learning about brand strategy through interactive experience.
